【题解】P3604 美好的每一天

作者: Sheep's Blog 日期: 2020-08-19 阅读时间:4.0 分钟 字数:807 分类: 莫队 题解 阅读量: -

这道题虽然三个月前就已经做过了，但是现在再看的时候，还是不会，于是决定写一篇题解来重新整理一下思路...

题面简述

给定一个字符串（只包含26个英文小写字母），每次询问一段区间内有多少个子区间满足里面的元素重排后能形成回文串。

首先我们考虑在什么情况下一段区间内的元素重排后能形成回文串：

1.当区间长度为奇数时，必定有一个字母的出现次数为奇数，其余字母均为偶数。

2.当区间长度为偶数时，必定所有字母的出现次数为偶数。

我们可以考虑利用异或的性质，

用000表示出现次数为偶数，111表示出现次数为奇数的情况。

那么显然可以得知，如果我们进行前缀和处理，设前iii位字母的出现情况为AiA_iAi​，区间[l,r][l,r][l,r]内字母的出现情况即为AlxorArA_lxorA_rAl​xorAr​。

考虑合法的情况只有AlxorArA_lxorA_rAl​xorAr​的值为2n2^n2n或000的情况，那么我们直接预处理出Aixor2nA_ixor2^nAi​xor2n的值，统计区间内Aixor2nA_ixor2^nAi​xor2n和AiA_iAi​出现了多少次即可。

然后就可以按照普通莫队的方法去做了。

代码如下：

#include <iostream>
#include <cstdio>
#include <algorithm>
#include <cmath>
using namespace std;
const int maxn=6e4+111,maxm=6e4+111;
int N,M,cnt,tot;
char S[maxn];
int ans[maxm];
int A[maxn],B[maxn],C[maxn],D[maxn][31],T[maxn];
struct query {
	int l,r,id;
	bool operator < (const query b) const {return B[l]!=B[b.l]?B[l]<B[b.l]:B[l]&1?r<b.r:r>b.r;}
}Q[maxm];
int pos(int x)
{
	int tt=lower_bound(C+1,C+cnt+1,x)-C;
	if(C[tt]==x) return tt;
	else return N+1;
}
#define lowbit(x) x&-x
void add(int x)
{
	for(int i=0;i<=25;i++) tot+=T[D[x][i]];
	tot+=T[D[x][26]];
	T[D[x][26]]++;
}
void del(int x)
{
	for(int i=0;i<=25;i++) tot-=T[D[x][i]];
	tot-=T[D[x][26]]-1;
	T[D[x][26]]--;
}
#define gc (getchar())
#define gs (ch<'0'||ch>'9')
void ri(int &x)
{
    x=0;char ch=gc;
    while( gs ) ch=gc;
    while(!gs ) x=x*10+ch-'0',ch=gc;
}
void wt(int x)
{
	if(x>9) wt(x/10);
	putchar(x%10+'0');
}
int main()
{
	ri(N),ri(M);
	scanf("%s",S+1);
	int block=sqrt(N);
	for(int i=1;i<=N;i++)
	{
		int x=S[i]-'a';
		A[i]=A[i-1]^(1<<x);
		C[i]=A[i];
		B[i]=(i+block-1)/block;
    }
    C[N+1]=0;
    sort(C+1,C+N+2);
    cnt=unique(C+1,C+N+2)-C-1;
    for(int i=0;i<=N;i++)
    {
   		for(int j=0;j<=25;j++) 
		   	D[i][j]=pos(A[i]^(1<<j));
   		D[i][26]=pos(A[i]);
    }
	for(int i=1;i<=M;i++) ri(Q[i].l),ri(Q[i].r),Q[i].l--,Q[i].id=i;
    sort(Q+1,Q+M+1);
	int L=1,R=0;
    for(int i=1;i<=M;i++)
    {
    	while(L<Q[i].l) del(L++);
    	while(L>Q[i].l) add(--L);
    	while(R<Q[i].r) add(++R);
    	while(R>Q[i].r) del(R--);
    	ans[Q[i].id]=tot;
    }
    for(int i=1;i<=M;i++) wt(ans[i]),putchar('\n');
    return 0;
}

时间复杂度O(26N+26MM)O(26N+26M \sqrt M)O(26N+26MM​) 。

好吧，即使之前做过，再做一遍的感觉和第一遍做没什么区别，全忘光了555。
