【题解】[NOI2010]能量采集

作者: Sheep's Blog 日期: 2020-08-16 阅读时间:8.0 分钟 字数:1424 分类: 题解 莫比乌斯反演 阅读量: -

题面简述：

给定NNN,MMM，求∑i=1N∑j=1M(gcd(i,j)∗2−1)\sum \limits_{i=1}^{N} \sum \limits_{j=1}^{M} (gcd(i,j)*2-1)i=1∑N​j=1∑M​(gcd(i,j)∗2−1),NNN,MMM<=1e5.

solution：

首先，显然，我们可以把"−1-1−1"提出来，那么我们只要求出∑i=1N∑j=1Mgcd(i,j)\sum \limits_{i=1}^{N} \sum \limits_{j=1}^{M} gcd(i,j)i=1∑N​j=1∑M​gcd(i,j)即可。

（下文中均假定N<=MN<=MN<=M)

方法一

我们考虑，当i∈[1,N],j∈[1,M]i∈[1,N],j∈[1,M]i∈[1,N],j∈[1,M]时，以ttt为公因数（不一定是最大公因数）的数对(i,j)(i,j)(i,j)有⌊Nt⌋⌊Mt⌋\lfloor\frac{N}{t}\rfloor\lfloor\frac{M}{t}\rfloor⌊tN​⌋⌊tM​⌋个（显然）。但考虑以ttt为最大公因数的数对时，就要消去kt(k>1,k∈Z)kt(k>1,k∈Z)kt(k>1,k∈Z)的影响。那么我们只需暴力枚举ktktkt即可，复杂度O(NN)O(N\sqrt{N})O(NN​)。

方法二

运用莫比乌斯反演的知识，将原式转换为带欧拉函数的式子求解，复杂度O(NN)O(N \sqrt{N})O(NN​)。

具体步骤如下：

f(d)=∑i=1N∑j=1M[gcd(i,j)==d]f(d)=∑i=1⌊Nd⌋∑j=1⌊Md⌋[gcd(i,j)==1]∵∑q∣xμ(q)=[x==1]∴f(d)=∑i=1⌊Nd⌋∑j=1⌊Md⌋∑q∣i,q∣jμ(q)f(d)=∑i=1⌊Nd⌋∑j=1⌊Md⌋∑q=1⌊Nd⌋μ(q)[q∣i,q∣j]f(d)=∑q=1⌊Nd⌋μ(q)∑i=1⌊Ndq⌋∑j=1⌊Mdq⌋f(d)=∑q=1⌊Nd⌋μ(q)⌊Ndq⌋⌊Mdq⌋f(d)=\sum \limits_{i=1}^{N} \sum \limits_{j=1}^{M}[gcd(i,j)==d]\\ f(d)=\sum \limits_{i=1}^{\lfloor \frac{N}{d} \rfloor} \sum \limits_{j=1}^{\lfloor \frac{M}{d} \rfloor} [gcd(i,j)==1]\\ ∵\sum_{q|x}\mu(q)=[x==1]\\ ∴f(d)=\sum \limits_{i=1}^{\lfloor \frac{N}{d} \rfloor} \sum \limits_{j=1}^{\lfloor \frac{M}{d} \rfloor}\sum_{q|i,q|j}\mu(q) \\ f(d)=\sum \limits_{i=1}^{\lfloor \frac{N}{d} \rfloor} \sum \limits_{j=1}^{\lfloor \frac{M}{d} \rfloor}\sum_{q=1}^{\lfloor \frac{N}{d} \rfloor}\mu(q)[q|i,q|j] \\ f(d)=\sum_{q=1}^{\lfloor \frac{N}{d} \rfloor}\mu(q)\sum \limits_{i=1}^{\lfloor \frac{N}{dq} \rfloor} \sum \limits_{j=1}^{\lfloor \frac{M}{dq} \rfloor}\\ f(d)=\sum_{q=1}^{\lfloor \frac{N}{d} \rfloor}\mu(q){\lfloor \frac{N}{dq} \rfloor}{\lfloor \frac{M}{dq} \rfloor}\\ f(d)=i=1∑N​j=1∑M​[gcd(i,j)==d]f(d)=i=1∑⌊dN​⌋​j=1∑⌊dM​⌋​[gcd(i,j)==1]∵q∣x∑​μ(q)=[x==1]∴f(d)=i=1∑⌊dN​⌋​j=1∑⌊dM​⌋​q∣i,q∣j∑​μ(q)f(d)=i=1∑⌊dN​⌋​j=1∑⌊dM​⌋​q=1∑⌊dN​⌋​μ(q)[q∣i,q∣j]f(d)=q=1∑⌊dN​⌋​μ(q)i=1∑⌊dqN​⌋​j=1∑⌊dqM​⌋​f(d)=q=1∑⌊dN​⌋​μ(q)⌊dqN​⌋⌊dqM​⌋

其实到这一步运用整除分块、预处理欧拉函数前缀和等等，要求的就是∑d=1Nf(d)×d\sum \limits_{d=1}^{N}f(d)×dd=1∑N​f(d)×d了。

但可以进一步优化：

∑i=1N∑j=1Mgcd(i,j)=∑d=1Nf(d)×d=∑d=1Nd∑q=1⌊Nd⌋μ(q)⌊Ndq⌋⌊Mdq⌋=∑d=1Nd∑dq=1Nμ(q)⌊Ndq⌋⌊Mdq⌋\sum_{i=1}^{N}\sum_{j=1}^{M}gcd(i,j)\\ =\sum_{d=1}^{N}f(d)×d\\ =\sum_{d=1}^{N}d\sum_{q=1}^{\lfloor \frac{N}{d} \rfloor}\mu(q){\lfloor \frac{N}{dq} \rfloor}{\lfloor \frac{M}{dq} \rfloor}\\ =\sum_{d=1}^{N}d\sum_{dq=1}^{N}\mu(q){\lfloor \frac{N}{dq} \rfloor}{\lfloor \frac{M}{dq} \rfloor}\\ i=1∑N​j=1∑M​gcd(i,j)=d=1∑N​f(d)×d=d=1∑N​dq=1∑⌊dN​⌋​μ(q)⌊dqN​⌋⌊dqM​⌋=d=1∑N​ddq=1∑N​μ(q)⌊dqN​⌋⌊dqM​⌋

设T=dqT=dqT=dq

=∑d=1Nd∑T=1N[d∣T]μ(Td)⌊NT⌋⌊MT⌋=∑T=1N⌊NT⌋⌊MT⌋∑d∣Tdμ(Td)=\sum_{d=1}^{N}d\sum_{T=1}^{N}[d|T]\mu(\frac{T}{d}){\lfloor \frac{N}{T} \rfloor}{\lfloor \frac{M}{T} \rfloor}\\ =\sum_{T=1}^{N}{\lfloor \frac{N}{T} \rfloor}{\lfloor \frac{M}{T} \rfloor}\sum_{d|T}d\mu(\frac{T}{d}) =d=1∑N​dT=1∑N​[d∣T]μ(dT​)⌊TN​⌋⌊TM​⌋=T=1∑N​⌊TN​⌋⌊TM​⌋d∣T∑​dμ(dT​)

设H(T)=∑d∣Tdμ(Td)H(T)=\sum\limits_{d|T} d\mu(\frac{T}{d})H(T)=d∣T∑​dμ(dT​)

由狄利克雷卷积的知识可得：

H=id∗μH∗1=id∗μ∗1∵ϵ=μ∗1∴H∗1=id∗ϵ∴H∗1=id∵ϕ∗1=id∴H=ϕH=id*\mu \\H*1=id*\mu*1 \\∵\epsilon=\mu*1\\ ∴H*1=id*\epsilon\\ ∴H*1=id\\ ∵\phi*1=id\\ ∴H=\phi H=id∗μH∗1=id∗μ∗1∵ϵ=μ∗1∴H∗1=id∗ϵ∴H∗1=id∵ϕ∗1=id∴H=ϕ

则原式即可化为∑T=1N⌊NT⌋⌊MT⌋ϕ(T)\sum\limits_{T=1}^{N}{\lfloor \frac{N}{T} \rfloor}{\lfloor \frac{M}{T} \rfloor}\phi(T)T=1∑N​⌊TN​⌋⌊TM​⌋ϕ(T)，那么运用整除分块即可求解。

方法三

其实对于上面得到的式子，我们有一个更快的推法：

我们考虑对于这个式子 ∑i=1N∑j=1Mgcd(i,j)\sum \limits_{i=1}^{N}\sum\limits_{j=1}^{M}gcd(i,j)i=1∑N​j=1∑M​gcd(i,j)

可以由n=∑d∣nϕ(d)n=\sum\limits_{d|n}\phi(d)n=d∣n∑​ϕ(d)推出

对于我们枚举的每个ddd，我们只用计算出它对于自己的倍数的贡献，加起来即可。

即

∑i=1N∑j=1Mgcd(i,j)=∑i=1N∑j=1M∑d∣gcd(i,j)ϕ(d)=∑d=1Nϕ(d)∑i=1N∑j=1M[d∣gcd(i,j)]=∑d=1Nϕ(d)∑i=1⌊Nd⌋∑j=1⌊Md⌋[1∣gcd(i,j)]=∑d=1Nϕ(d)⌊Nd⌋⌊Md⌋\sum \limits_{i=1}^{N}\sum\limits_{j=1}^{M}gcd(i,j)\\ =\sum \limits_{i=1}^{N}\sum\limits_{j=1}^{M}\sum\limits_{d|gcd(i,j)}\phi(d)\\ =\sum\limits_{d=1}^{N}\phi(d)\sum \limits_{i=1}^{N}\sum\limits_{j=1}^{M}[d|gcd(i,j)]\\ =\sum\limits_{d=1}^{N}\phi(d)\sum \limits_{i=1}^{\lfloor\frac{N}{d}\rfloor}\sum\limits_{j=1}^{\lfloor\frac{M}{d}\rfloor}[1|gcd(i,j)]\\ =\sum\limits_{d=1}^{N}\phi(d){\lfloor\frac{N}{d}\rfloor}{\lfloor\frac{M}{d}\rfloor} i=1∑N​j=1∑M​gcd(i,j)=i=1∑N​j=1∑M​d∣gcd(i,j)∑​ϕ(d)=d=1∑N​ϕ(d)i=1∑N​j=1∑M​[d∣gcd(i,j)]=d=1∑N​ϕ(d)i=1∑⌊dN​⌋​j=1∑⌊dM​⌋​[1∣gcd(i,j)]=d=1∑N​ϕ(d)⌊dN​⌋⌊dM​⌋

void ask_phi()
{
	phi[1]=1;
	for(int i=2;i<=N;i++)
	{
		if(!phi[i]) //i为质数时
		{
			phi[i]=i-1;
			for(int j=i+i;j<=N;j+=i)
			{
				if(!phi[j]) phi[j]=j;
				phi[j]=phi[j]/i*(i-1);
			}
		}
		pre[i]=pre[i-1]+phi[i];
	} 
}
