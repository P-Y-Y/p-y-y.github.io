对拍

受zlt dalao所托，帮他写了一份对拍程序，感觉很有用，一年前没认真学。
在此记录一下：

#include <bits/stdc++.h>
using namespace std;
int main()
{
	int tt=0;
	while(++tt)
	{
		system("maker.exe");//造数据的程序
		system("1.exe");//程序1
		system("2.exe");//程序2
		printf("Case %d: ",tt);//第tt组数据
		if(system("fc 1.out 2.out"))//如果没有写文件输出，那么对拍的时候是可以看到两个程序的输出的
		{
			printf("WA");
			break;
		}
		printf("AC\n");
	}
	return 0;
}

造数据

（以下部分引自lcf学长1年前讲课的课件）
随机负数：先随机绝对值，在通过一个随机数的奇偶性来确定正负。
随机随机区间：随机两个端点，取较小值为左端点，较大值为右端点，如果二者相等有必要的话要再随机一遍。
随机一棵树：给每个点找一个父亲。注意随机树的树高期望是O(logn)O(log n)O(logn) 的，所以我们可能需要先手动构造出一组长度为 O(n)O(n)O(n)的链，再来随机找父亲。
随机一张图：先随机出一棵树，再随机加入一些边。
随机一个排列：对于每个数 aia_iai​，在区间[1,i][1, i][1,i] 内随机一个数 kkk，交换 aia_iai​，aka_kak​ 的值。（其实也可以直接用 <algorithm>中的函数random_shuffle(a+1,a+n+1)。）
