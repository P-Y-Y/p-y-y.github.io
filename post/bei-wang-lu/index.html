巧妙的函数/写法

  1. n^=m^=n^=m 这个可以当成swap(n,m)来写。

  2. 快速乘：

    有时我们直接乘会爆long long，那么我们可以用快速乘来解决。

    写法和思想可以类比快速幂：

    ll quickmul(ll a,ll b)
{
    ll res=0;
    while(b)
    {
        if(b&1) res+=a;
        a=(a+a)%M; 
        b>>=1ll;
    }
    return res;
}

  3. x-y>>31等价于x<yx<yx<y（注意x−yx-yx−y要在int范围内），且这样写可以快333~555倍（来自fyk的小技巧）。

  4. __builtin_popcount(x)快速统计xxx的二进制表示中有多少个111，注意前面是两个下划线。

  5. 亲测：C++自带的min max函数比你自己写的快多了，但要注意两个作比较的变量的类型要相同。

做题技巧

  1. 普通DP想不出来的时候可以考虑考虑权值存的状压DP。

  2. 每个节点度数不超过2不仅仅只有链的情况，还有环的情况。

  3. 会打优秀的暴力也是一种能力。

  4. 做DP的时候，如果转移的状态整体来说比较小，可以考虑矩阵快速幂优化（例：2020NOID1T1）
